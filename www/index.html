Présentation

Le catalogue nos livres recense les livres électroniques du domaine public francophone, disponibles gratuitement et qui ont un minimum de qualité.

Les sites suivant sont recensés :

  • Bibliothèque électronique du Québec (BEQ)
  • Bibliothèque numérique romande (BNR)
  • Bibliothèque Russe et Slave (BRS)
  • Chine ancienne
  • Ebooks Libres et Gratuits (ELG)
  • ÉFÉLÉ
  • Gallica
  • La bibliothèque de Gloubik
  • Meskach
  • Mobile Read - roger64
  • Projet Gutenberg (livres en français)
  • Rousseau Online
  • Toute petite bibliothèque numérique de Belgique (TPBNB)
  • Wikisource francophone

Merci à eux de partager leurs efforts.

Le catalogue est mis à jour environ une fois par semaine.

Vos commentaires sont bienvenus : contact@noslivres.net.

Mode d’emploi

Pour trier sur une colonne, cliquez sur les flèches dans l’en-tête de la table.

Pour trier sur plusieurs colonnes, cliquez sur les flèches de la première colonne, puis shift-cliquez sur les flèches des colonnes secondaires.

Pour chercher des livres, entrez des mots ou fragments de mots dans la boite Rechercher. Chaque mot ou fragment doit être présent dans une entrée, mais ils ne doivent pas forcément être tous dans la même colonne.

Pour accéder à un livre, cliquez sur le lien dans la colonne site.

Le à la droite de la boîte de recherche est un lien sur cette page, initialisée avec les termes de la recherche courante.

Ajouter votre site

Chaque site recensé fournit son catalogue, à une adresse web publique, sous l’une de ces trois formes :

  • XML/RDF, tel que produit par le projet Gutenberg (voir http://www.gutenberg.org/wiki/Gutenberg:Feeds pour une description de ce format)
  • OPDS
  • un fichier texte. Une ligne par livre, une ligne d'en-tête. Chaque ligne est faite de cinq colonnes, séparées par le caractère TAB (U+0009) :
    1. auteur (nom, prénom)
    2. titre
    3. URL d’une page de votre site sur laquelle on peut trouver le livre dans une ou plusieurs formats
    4. date de parution, dans le format ISO 8601 (par exemple, 2013-11-30, pour le 30 novembre 2013)
    5. date de dernière mise à jour, dans le même format
    Le fichier doit être en UTF-16. Ce format correspond au format « Texte Unicode » de Excel, ou bien au format « Texte CSV » de Libre Office, (jeu de caractères « Unicode », séparateur de champ « (Tab) »). Voir http://efele.net/ebooks/efele_catalogue_commun.txt pour un exemple de ce format.

Lorsque votre catalogue est prêt, informez nous (contact@noslivres.net) pour que nous l'intégrions.

noslivres se réserve le droit de refuser de recenser un site.
